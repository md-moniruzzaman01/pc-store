
# Computer Parts eCommerce Project

A modern web application for buying and exploring computer parts, built with **Nextjs**, **React Router**, and **Tailwind CSS**. This project provides a smooth and responsive user experience with dynamic pages, transitions, and modular architecture.

---

## Features

* **Responsive Layout**: Fixed Navbar with SearchBar and Category section.
* **Multiple Pages**:

  * Home
  * Products
  * About Us
  * Contact Us
  * 404 Not Found
* **Smooth Animations**: Fade and slide transitions for better UX.
* **Product Browsing**: Explore products by category.
* **Contact Form**: Users can submit inquiries.
* **Modular and Scalable**: Easy to extend and maintain.

---

## Tech Stack

* **Frontend**: Nextjs, React Router, Tailwind CSS

## Installation

```bash
# Clone the repository
git clone https://github.com/md-moniruzzaman01/pc-store.git

# Navigate to project folder
cd computer-parts-ecommerce

# Install dependencies
npm install

# Start the development server
npm start
```

---

## Usage

* Navigate through pages using the Navbar.
* Search products using the SearchBar.
* Submit inquiries via the Contact Us page.
* Explore products by category.
